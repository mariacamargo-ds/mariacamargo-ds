Meu nome é Maria Eduarda, sou estudante de Ciência de Dados na **Fatec Cotia** e bolsista de Iniciação Científica no programa **SmartB100**, pela FAPESP (Fundação de Amparo à Pesquisa no Estado de São Paulo).
